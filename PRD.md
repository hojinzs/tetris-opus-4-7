# Tetris Game — PRD

## 1. 개요

### 1.1 제품 비전
브라우저에서 즉시 실행 가능한 **클래식 픽셀/레트로 스타일의 싱글 플레이어 테트리스**. 모던 테트리스의 스코어링 시스템(T-Spin, 콤보, Back-to-Back)과 로컬 리더보드를 통해 반복 플레이와 기록 경쟁의 재미를 제공한다.

### 1.2 목표
- 웹 브라우저에서 별도 설치 없이 플레이 가능
- 표준 테트리스 조작감과 모던 스코어링 규칙 준수
- 레트로 감성의 시각 스타일 + 사운드 이펙트
- 개인 최고 기록을 로컬에 저장하여 도전 의욕 제공

### 1.3 대상 사용자
- 테트리스 클래식 팬
- 브라우저에서 짧게 즐길 캐주얼 게임을 찾는 사용자
- 자기 기록 경쟁을 즐기는 플레이어

## 3. 게임플레이 요구사항

### 3.1 보드
- **플레이필드**: 10칸(너비) × 20칸(높이, 가시 영역) + 상단 2칸(숨김 버퍼)
- **셀 크기**: 30px 기준 (반응형 스케일)
- **블록**: 7종 표준 테트로미노 (I, O, T, S, Z, J, L)

### 3.2 피스 생성
- **7-Bag 랜덤화**: 한 세트에 7종이 정확히 한 번씩 등장하도록 셔플된 가방 사용

### 3.3 조작 (표준 매핑)
| 키 | 동작 |
| --- | --- |
| ← / → | 좌/우 이동 |
| ↓ | 소프트 드롭 |
| Space | 하드 드롭 |
| ↑ / X | 시계 방향 회전 |
| Z / Ctrl | 반시계 회전 |
| Shift / C | Hold |
| P | 일시정지 |
| Esc | 메뉴로 |
| R | 재시작 (게임오버 후) |

- **DAS (Delayed Auto Shift)**: 170ms
- **ARR (Auto Repeat Rate)**: 50ms
- **Soft Drop 속도**: 중력의 20배

### 3.4 회전 시스템
- **SRS (Super Rotation System)** 적용
- 벽차기(Wall Kick) 테이블: SRS 표준 (I 피스 별도 테이블)

### 3.5 모던 기능
- **Hold**: 한 피스를 홀드 슬롯에 보관. 피스 고정 전까지 한 번만 사용 가능
- **Next Queue**: 다음 5개 피스 미리보기
- **Ghost Piece**: 하드 드롭 시 놓일 위치를 보드 바닥에 반투명으로 표시
- **Lock Delay**: 착지 후 0.5초 유예, 이동/회전 시 리셋 (최대 15회)

## 4. 스코어링 & 레벨 (모던 방식)

### 4.1 레벨 진행
- **시작 레벨**: 1 (설정에서 1~15 선택 가능)
- **레벨업 조건**: 매 10줄 클리어마다 +1 레벨
- **중력 (초당 셀 이동)**: 공식 `G = (0.8 - (level-1)*0.007)^(level-1)` 기반

### 4.2 라인 클리어 점수
| 라인 | 기본 점수 |
| --- | --- |
| Single | 100 × level |
| Double | 300 × level |
| Triple | 500 × level |
| Tetris | 800 × level |

### 4.3 T-Spin
| 동작 | 점수 |
| --- | --- |
| T-Spin (no lines) | 400 × level |
| T-Spin Single | 800 × level |
| T-Spin Double | 1200 × level |
| T-Spin Triple | 1600 × level |
| T-Spin Mini | 100 × level |
| T-Spin Mini Single | 200 × level |

### 4.4 Back-to-Back (B2B)
- Tetris 또는 T-Spin 라인 클리어를 연속으로 성공 시 해당 점수 × 1.5
- 일반 라인 클리어로 끊어짐

### 4.5 콤보
- 연속 라인 클리어 시 `50 × combo × level` 추가
- 라인 클리어 실패 시 리셋

### 4.6 기타 점수
- Soft Drop: 이동 셀당 1점
- Hard Drop: 이동 셀당 2점
- Perfect Clear (보드 완전 비움) 보너스:
  - Single: 800 × level
  - Double: 1200 × level
  - Triple: 1800 × level
  - Tetris: 2000 × level

## 5. 화면 & UI

### 5.1 화면 구성
```
┌──────────────────────────────────────────┐
│                 TETRIS                   │
├────────┬───────────────────┬─────────────┤
│  HOLD  │                   │    NEXT     │
│ [___]  │                   │   [___]     │
│        │                   │   [___]     │
├────────┤   PLAY FIELD      │   [___]     │
│ SCORE  │    (10 × 20)      │   [___]     │
│ LEVEL  │                   │   [___]     │
│ LINES  │                   │             │
│ TIME   │                   │             │
└────────┴───────────────────┴─────────────┘
```

### 5.2 화면(씬)
1. **타이틀/메인 메뉴**: Start, Leaderboard, Settings, How to Play
2. **게임 플레이 화면**
3. **일시정지 오버레이**: Resume, Restart, Main Menu
4. **게임 오버 화면**: 최종 점수, 신기록 표시, 이름 입력(3글자), Restart, Main Menu
5. **리더보드 화면**: 상위 10개 기록 (Rank, Name, Score, Lines, Level, Date)
6. **설정 화면**: 시작 레벨, 음량(BGM/SFX), 고스트 ON/OFF
7. **How to Play 화면**: 키 매핑, 규칙 안내

### 5.3 시각 스타일 — 클래식 픽셀/레트로
- NES 스타일 도트 그래픽
- 블록 색상: 표준 테트리스 컬러 팔레트
  - I=시안, O=노랑, T=보라, S=초록, Z=빨강, J=파랑, L=주황
- 각 블록에 픽셀 테두리 및 내부 하이라이트/그림자로 3D 느낌
- 픽셀 폰트 사용 (예: `Press Start 2P`)
- CRT 스캔라인 효과 (옵션 토글)

### 5.4 애니메이션 / 이펙트
- 라인 클리어 시 플래시 → 제거 애니메이션 (약 300ms)
- Tetris / T-Spin 성공 시 화면 흔들림 + 텍스트 팝업
- 레벨업 시 배경 색상 변화
- 게임 오버 시 보드가 위에서부터 회색으로 채워지는 효과

## 6. 사운드

- 피스 이동 / 회전 / 착지 / 하드 드롭

## 7. 리더보드 (로컬)

### 7.1 저장 방식
- `localStorage` 키: `tetris_leaderboard_v1`
- 상위 10개 기록만 유지

### 7.2 저장 데이터 구조
```json
{
  "entries": [
    {
      "name": "AAA",
      "score": 125000,
      "lines": 142,
      "level": 15,
      "durationSec": 612,
      "date": "2026-04-17T10:23:00Z"
    }
  ]
}
```

### 7.3 신기록 처리
- 게임 오버 시 점수가 기존 10위 안에 들면 3글자 이니셜 입력 UI 표시
- 입력 완료 시 목록에 삽입 후 정렬 → 상위 10개만 유지

## 8. 게임 오버 조건
- 새 피스를 스폰 위치(상단 버퍼)에 배치할 수 없을 때
- 피스가 보드 상단 경계를 넘어 고정될 때 (Lock Out)

## 12. 성공 지표
- 주요 브라우저에서 60fps 유지
- 입력 지연(키 누름 → 화면 반영) 50ms 이하
- SRS 회전/벽차기가 표준 가이드라인과 100% 일치
- 한 게임 세션 평균 5분 이상 유지
